# 🎭 Persona Bot

A simple AI chatbot built with **Streamlit** and **LangChain Groq** that allows users to chat with an AI having any personality they choose.

## 🚀 Features

- 🤖 Create any AI persona
- 💬 Interactive chat interface
- 🔑 User enters their own Groq API Key
- 🔄 Reset chat with a new personality
- ⚡ Powered by Groq's Llama 3.3 70B model
- 🎨 Clean Streamlit UI

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/persona-bot.git
```

Move into the project folder

```bash
cd persona-bot
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the App

```bash
streamlit run app.py
```

---

## 🔑 Groq API Key

Create a free API key from

https://console.groq.com/keys

Paste the API key into the sidebar when the application starts.

---

## 💡 How it Works

1. Enter your Groq API Key.
2. Define any AI personality.
3. Click **Reset Chat & Apply Persona**.
4. Start chatting with your custom AI.
